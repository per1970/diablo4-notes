# Guide – Talisman: Perfecting Charms & Seals

> Source: wudijo (YouTube, May 2, 2026) + Fextralife / IcyVeins / Boostmatch.gg cross-reference

## Overview

The Talisman is a separate inventory tab unlocked by completing the *Last of the Horadrim* questline in the Lord of Hatred campaign. It has 7 slots: one central **Seal** and up to six outer **Charm** slots. It is permanent expansion content — not a seasonal mechanic — and will outlive Season 13.

Charms drop freely in higher Torment content. At endgame, filtering out non-Ancestral charms reduces clutter while still providing plenty of upgrade material.

---

## Seals

### What to Look For
- **+1 Charm Slot** — the single most important affix; gives you the 6th slot for a Unique Charm
- A large **damage multiplier** affix (e.g. 16%+ damage multi)
- Bonus: a Seal affix that synergizes with your active set (rare, but high value)

### How to Get the 6th Slot
Seals cannot be individually rerolled for slot count. The only method is the **3-to-1 Horadric Cube recipe**: feed three Seals into the cube to produce one new Seal. Estimated ~1 in 50 chance to roll a +1 charm slot result. Time-consuming, but the only path.

### Seal Affix Pool (general)
- Damage multiplier (highest value)
- All Stats
- Life
- Movement Speed
- Skill Ranks (build-dependent)

## Set Charms

### How to Target a Specific Set Piece
Any Set Charm can be converted into a different piece of the same set via the **Roll Set Charm** recipe in the Horadric Cube. There are 5 different set slot types per set (e.g. Phoba, Lore, etc.). This is the primary method to complete a set — no need to wait for the exact piece to drop.

> Rolling can also produce a Greater Affix (GA) on Ancestral charms, so it's worth doing even purely for stat improvement.

Set Charms only begin dropping at **Torment I** and above.

### Ideal Stat Targets

| Priority | Affix | Notes |
|---|---|---|
| Primary ×4 | +3 to a skill category (e.g. Occult, Hellfire) | Nearly doubles skill ranks with a full 5-piece set |
| Secondary ×2 | All Stats | Activates Paragon nodes; provides life + minor bonuses |
| Situational | Resources | Relevant for resource-hungry classes (e.g. Barbarian) |
| Situational | Movement Speed | Only if not already capped |

**Target loadout: 4× Skill Ranks (+3 each) + 2× All Stats.**

With a 5-piece set + 4 charms at +3 ranks each = **+12 skill ranks total**, effectively doubling your ranks.

All Stats value: slightly weaker raw damage than pure skill ranks, but it activates Paragon advantage nodes — especially relevant in S13 now that Shroud of False Death is no longer universal and All Stats is harder to source elsewhere.

## Unique Charms

### How to Craft
In the Horadric Cube under **Item Transmutation → Craft Unique Charm**:
- 1× Ancestral version of the target Unique
- 3× any random Uniques (used as fodder)
- The Unique's power carries over; attributes/affixes are **randomized on creation**

### Critical Caveats
- **Do not reroll the Unique Power before converting** — the power value is re-randomized after transmutation anyway; rerolling beforehand wastes materials
- **Two-handed items**: power value is halved when converted to a charm (same scaling rule as aspects)
- **Amulets**: the inherent 100% bonus is removed on conversion — no oversized amulet charms
- **Cannot reroll after creation** — the roll is final; craft again to try for better
- **Cannot double-equip**: if the Unique is already equipped on your character, the Unique Charm version cannot be slotted simultaneously

### Target Slot
Unique Charms go into the **6th charm slot** — the one only accessible after getting a +1 slot Seal. The first 5 slots fill with your 5-piece Set Charms; the 6th is exclusively for a Unique Charm.

### High-Value Candidates
- Banish Lord's Talisman
- Red Blessing
- Spiritborn key-passive amulets
- Any build-defining Unique that currently locks a gear slot

**Impact:** roughly 50–100% extra DPS for most builds depending on the Unique chosen.

## Mythic Seals

The endgame chase item for the Talisman. Allows equipping a **full 5-piece Set bonus AND a 3-piece Set bonus simultaneously** — impossible with any standard Seal.

- Can also roll random affixes like standard Legendary Seals
- 3 Mythic Seals exist in total; only one (the 5+3 combo) is currently considered build-viable — the other two are assessed as too weak for practical builds
- Extremely rare: wudijo found 1 Mythic Seal across approximately 20 normal Mythic drops

### Drop Sources
| Source | Notes |
|---|---|
| **Uber Meisto** (Crux of the False Prophet key) | Only boss with explicit Mythic Seal drop tooltip — likely the intended primary source |
| **Echoing Hatred** (wave survival) | Confirmed drop (wudijo personal drop) |
| **Undercity Tributes** with extra Charm rewards | Targeted farming option |
| **War Plan** activities with charm rewards | Occasional source |

## Gold Income Tip

Once your Talisman is fully optimized, excess Seals and Charms can be vendored at the **Purveyor of Curiosities** for meaningful gold income:
- Legendary Seals: ~13.3 million gold each
- Set Charms: ~1.5 million gold each

With Enchanting costs being very high in S13, rendering surplus Seals and Charms is recommended over cube-rolling once your setup is locked in.

## Quick Checklist

- [ ] Complete *Last of the Horadrim* questline to unlock Talisman
- [ ] Farm/cube a Legendary Seal with +1 Charm Slot + damage multi affix
- [ ] Complete your 5-piece Set via Roll Set Charm in Horadric Cube
- [ ] Optimize charm stats: 4× +3 Skill Ranks + 2× All Stats
- [ ] Craft target Unique Charm (Ancestral item + 3 fodder Uniques; accept RNG on power roll)
- [ ] Farm Uber Meisto (Crux of the False Prophet) for a Mythic Seal
