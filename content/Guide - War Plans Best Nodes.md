# War Plans – Best Node Paths (Season 13)

> **Sources:** wudijo (YouTube, May 1 2026) · Mobalytics · Keengamer · Maxroll · Vortex Gaming

War Plans is a **permanent LoH expansion feature** — not seasonal. Unlocked after completing the Lord of Hatred campaign and reaching Torment difficulty. Access via the Command Table in **Temis** (Skovos).

## ⚠️ Key Warnings Before Spending Points

- Respec costs scale steeply — ~46–50M gold per point at higher ranks
- Nodes only activate when that activity is **included in an active War Plan run**
- You start with ~7 usable points per tree; focus depth over breadth
- **Pit tree** requires Season Rank 2 (clear Hellish Descent capstone) to unlock
- **Lair Boss Greater nodes** require Torment 6+
- XP is the hardest resource to gain in S13 — prioritize XP nodes across trees where relevant

## Tree Recommendations

### 🔥 Helltide — S Tier
**Best Path: Middle**

- Hellwyrm ambushes spawn Maggots → drop massive XP orbs (20–30M XP per worm on T12)
- Also drops **Lair Keys** and Boss Trophies (turn in at Horadric Cube for Uniques)
- Considered the **best XP farm** in the game when properly set up
- Left path offers Profane Mindcages (harder + more cinders/XP) — secondary option
- Hellborn Butchers (right side) are risky but rewarding with leftover points if survivability is solid

---

### 🕳️ Nightmare Dungeons — B Tier
**Best Path: Right (general) or Middle (Astaroth farming)**

- **Right:** Treasure Goblins + Treasure Breaches — high chance at rare loot and Mythics; can self-sustain Breaches via rotations
- **Middle:** Infinite Escalating Nightmare Sigils via *Recurring Nightmare* node — required if farming Astaroth for specific loot table drops (e.g. Tyrant's Grasp ring)
- **Left:** Shrine-based elite respawn mechanic — decent if combined with Nemesis, nothing crazy
- Tuned Primordial Dust available in this tree — only source; useful if rerolling Unique affixes heavily

---

### 🏙️ Undercity — A Tier
**Best Path: Left**

- Invasion Portals + Portal Pranksters → bonus chests including **Forgotten Souls** (most valuable crafting mat)
- Leftover Tributes node: dump small tributes at end for direct chest loot — skips full runs
- Put 1 point in XP orb node — experience carries across all characters

---

### ⚔️ The Pit — B Tier
**Best Path: Middle + Chrono's Haste**

- **Chrono's Haste** (sub-5 min = +2 glyph upgrades/run) — first priority pickup
- **Heart of Stone** mechanic: pick up item at run start, keep it to the end → +1 glyph upgrade + more progression orbs → shaves 20–30 seconds per run
- **Middle path XP orbs** stack with Heart of Stone orbs; ~10M XP dumped at end of run
- Pit = ~5M+ base XP per run + 10M from orb dump at end — solid consistent farm for fast builds (2–3 min runs)
- **Right side (Artificer/Cursed nodes) — rated poor:** shows up rarely, hard to notice, not worth the investment

---

### 💀 Lair Bosses — A Tier
**Best Path: Middle (Nemesis Lair)**

- Chained boss encounters: kill → dual fight → triple fight → massive loot
- Community reports easy Mythic drops from Nemesis Lair chain
- Requires investment in boss kills to unlock, but high payoff
- Pair with whichever boss loot table you need (e.g. Vashans via Infernal Hordes path, Astaroth via NMD path)

---

### 👿 Infernal Hordes — B Tier
**Key Node: Method on Madness**

- Locks in a specific Infernal strategy (Soulspire, Lords, Masses) reliably each run
- Combine with *Additional Infernal Offering* for strong synergy — but blocks Chaos Waves
- If you want Vashans from Chaos Waves (via Lair Boss tree), **don't block Chaos Waves**
- Right side generally considered weak — limited testing at this stage

---

### 🌳 Tree of Whispers — C Tier (Passive)
**Best Path: Middle (Material Cache) or right (Spark)**

- Material Cache gives Forgotten Souls, gold, mats — useful in late endgame when gear matters less
- Mystery Cache (deep path) can yield a Resplendent Spark — appealing while Mythics are scarce
- Whispers run concurrently with your War Plan and grant passive tree XP — no dedicated run needed
- Lower overall investment priority; spec reactively

---

## General Strategy

- Pick **1–2 activities** to spec deep into based on your current farming goal
- A fully specced tree vastly outperforms three half-built trees
- **Early game:** Helltide middle + Undercity left = XP + Forgotten Souls foundation
- **Glyph push:** Pit with Chrono's Haste + Heart of Stone
- **Mythic hunting:** Lair Boss Nemesis chain or NMD right (Goblins/Treasure Breaches)
- Endgame goal: fully unlock all trees → run juiced single-activity sessions (e.g. 1hr straight Helltide)
